# LucaWriter v1.1.0

发布日期：2026-05-16

## 🎨 UI 大改造

- **Luca 聊天框从右侧移到左侧**：整体布局改为 trigger(18px) → leftStack(聊天+章节列表overlay) → resizeHandle → editor-area
- **章节列表 overlay 模式**：章节列表以 absolute overlay 覆盖在聊天框上方，鼠标移入秒开、移出秒关
- **底栏布局重构**：AI 输入框在左边（与聊天框宽度同步），Tab 栏在右边；Grid 布局物理约束行高，彻底解决底栏溢出问题
- **编辑器 topbar 整合**：章节标题、字数统计、"本章写完"按钮从编辑器顶部移到全局 topbar
- **深色/浅色模式切换**：书架和系列视图新增深色模式切换按钮和设置按钮
- **底栏 Tab 按钮重新设计**：增加间距、背景色、hover/active 状态优化
- **通读面板"思考中"动画优化**：三个独立元素波浪弹跳，替代原来的 box-shadow 伪点方案
- **弹窗交互修复**：选中文本拖到窗口外不再误关闭弹窗

## 🤖 统一 Luca 对话架构

- **一个 Luca**：所有窗口共享对话历史，除非手动清除，否则在任何地方访问都带有之前的记录
- **浏览位置提示**：系统提示词告知 Luca 当前焦点（系列-书本-章节名），不注入正文
- **[READ_CHAPTER] 工具**：Luca 需要正文时自主调用工具读取，支持一次最多 3 个章节
- **自动上下文压缩**：对话结束后 30 秒闲时压缩，AI 生成摘要替代简单截断，保留最近 12 条消息
- **焦点公告**：切换系列/书本/章节时，聊天框显示当前焦点位置
- **旧历史惰性迁移**：首次读取全局历史时自动合并各书/系列目录下旧对话记录

## 🔍 知识库核对（吃书雷达）

- **AI 设定纠错**：AI 聊天中可引用知识库出处（[CITE]）、提议修改设定（[PROPOSE_KB_EDIT]）
- **Citation 卡片**：显示引用来源，支持跳转到对应章节并闪烁高光
- **Proposal 弹窗**：修改提议以弹窗形式展示，含 diff 显示和确认/拒绝按钮
- **主动提议规则**：AI 在发现矛盾、用户纠正、信息过时等情况下必须主动提议修改
- **"你去确认"深度分析**：吃书雷达条目新增深度确认按钮，AI 引用原文出处分析矛盾点

## 📝 编辑器增强

- **搜索与替换**：Ctrl+F 打开搜索、Ctrl+H 打开替换、F3/Ctrl+G 导航匹配项
- **搜索高亮与标注共存**：搜索匹配作为独立事件类型参与高亮排序，不干扰标注
- **实体查找**：大纲面板中实体/设定/事件的🔍按钮，点击直接在编辑器中定位
- **独立字号调整**：编辑器字号与全局字号分离，7 档可选（14-28px），自动持久化
- **流式回复滚动修复**：流式输出时只替换当前消息 DOM，不清空整个容器，不再抽搐

## 📊 时间线可视化

- **时间长河设计**：细线脊骨+流光动画+彩色左边框泡泡+事件点+连接茎+章节标记
- **类型着色**：人物=青、地点=琥珀、规则=紫、物品=橙、其他=绿
- **点击跳转**：点击泡泡跳转到对应章节并定位原文片段（优先使用 evidence 原文引用）
- **水平滚轮**：鼠标滚轮改为水平滚动，支持触摸板横滑

## 🧠 通读流水线重构

- **流式调用**：通读过程改为流式 API 调用，实时反馈进度
- **自适应批次**：根据内容长度自动调整每批处理的章节数
- **文件日志**：通读过程写入文件日志，便于排查问题

## 📡 SSE 推送

- **AI 活跃指示器**：从轮询改为 SSE 推送，实时显示 AI 任务状态
- **自动重连**：连接断开后 3 秒自动重连，30 秒无消息发送 keepalive

## 🔧 其他改进

- **AI 多轮对话修复**：多轮对话时 Luca 现在也能看到当前章节内容
- **预言手动更新修复**：知识库变动后手动更新预言不再报 not found
- **浏览器自动填充禁止**：AI 输入框不再弹出浏览器自动填充建议
- **系列视图面包屑**：移除多余的 › 分隔符
- **封面图饱和度降低 50%**
- **UI 缩放设置**：新增 ui_scale 设置和 theme-is-light API

# LucaWriter v1.2.1

发布日期：2026-05-31

## 💬 Luca 聊天体验优化

- **流式更新不再销毁 DOM**：AI 回复流式输出时引用卡片、主题卡片等不再每 600ms 被销毁重建
- **历史面板移到横条下方**：点击历史对话不再挤占品牌栏空间
- **自动对话标题**：会话标题从第一条用户消息自动生成，显示在 Luca 旁
- **顶栏死限位**：拖拽弹出窗口时顶部小横条不会越过顶栏下边界
- **历史面板溢出修复**：展开历史面板时动态计算可用空间，不再把内容推出屏幕外
- **消息区 `min-height` 移除**：允许 Flexbox 完全压缩，不再因消息区卡死导致溢出
- **`popup-messages` `min-height: 60→0`**：解决 flex 布局负空间分配问题
- **毛玻璃浮标通知**：AI 完成回复且聊天框收起时，在输入框上方显示 6 秒消息气泡（毛玻璃+左对齐）
- **鼠标移开立即收起**：关闭延迟从 350ms 改为 0ms
- **误触浏览器边缘不收起**：鼠标移向窗口外部不再触发弹出窗口关闭

## 🐛 其他修复

- **修复 `_localLLMSpeed` 未定义**：添加缺失的变量声明
- **修复 `memoryUpdating` 未定义**：添加缺失的变量声明

## 🎯 内置模型策略收紧

- **仅 NVIDIA 显卡可用**：内置本地模型（llama.cpp）现仅支持检测到 NVIDIA 显卡且显存 ≥8GB 时启用
- **禁用 Mac / AMD / 纯 CPU 路径**：Apple Silicon、AMD 显卡、纯 CPU 推理路径暂时关闭，统一降级为 API 兜底
